pm2

2016/11/6 9:30 上午 posted in  nodejs comments  

nodejs 的进程管理器，自带负载均衡以及热重载

全局安装

$ npm install pm2 -g

开机启动

pm2 save 保存当前进程列表，下次启动时自动恢复列表内进程

$ pm2 startup && pm2 save

启动

[ app_name ] : 应用名
[ 4 ] :进程数目,为0时进程数目与 CPU 数相等

$ pm2 start app.js --name=[app_name] -i 4

暂停 | 删除

[ id ] : 进程 id

$ pm2 stop [id]
$ pm2 delete [id]

进程数变为2

[ app_name ] : 应用名

$ pm2 sacle [app_name] 2

监控

$ pm2 list
$ pm2 monit
$ pm2 show [app_name]
$ pm2 logs
$ pm2 logs [app_name]

此外，还可以通过 keymetrics 实现网页监控，不过免费版只能支持一台机器的状态监控
