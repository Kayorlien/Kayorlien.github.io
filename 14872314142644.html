[译] Sequelize(4) 范围

2016/11/27 10:49 上午 posted in  Sequelize comments  

范围（scope）: 定义好范围,在后续的操作中可以减少选择条件的重复，比如，只在未删除用户列表中进行操作。

范围是对模型的一种分类

// 修改默认所有操作的范围 default Scope
var Project = sequelize.define( 'project', {
    // Attributes
}, {
    defaultScope: {
        where: {
            active: true
        }
    },
    scopes: {
        deleted: {
            where: {
                deleted: true
            }
        },
        activeUsers: {
            include: [
                { model: User, where: { active: true } }
      ]
        }
    }
} );



// 移除默认范围，设置为 deleted 范围
Project.scope( 'deleted' ).findAll();

// 与上面等价
var DeletedProjects = Project.scope( 'deleted' );
DeletedProjects.findAll();

var Post = sequelize.define( 'post', attributes, {
    defaultScope: {
        where: {
            active: true
        }
    },
    scopes: {
        deleted: {
            where: {
                deleted: true
            }
        }
    }
} );

例子

User.hasMany( Post ); // regular getPosts association
User.hasMany( Post.scope( 'deleted' ), { as: 'deletedPosts' } );
