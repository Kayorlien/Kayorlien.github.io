css 命名约定

2017/3/22 14:24 下午 posted in  web comments  

Object Oriented CSS (OOCSS)

分离容器与内容

别这么做啊！！！！

ul li.list-item {
    ...
}

直接跳到元素

.list-item {
    ...
}

Atomic CSS (ACSS)

将每个元素拆分成原子

.mt-20 {
    margin-top : 20px;
}

.fl {
    float : left;
}

Block, Element, Modifier (BEM)

根据严规写出的，易于理解的 css 风格

/* 块 */
.block {}

/* 块中元素，双下划线 */
.block__element {}

/* 元素或块的修改器，双连字符 */
.block--modifier {}

一个例子

<header class="col-header">

    <!-- 块元素 -->
    <h1 class="col-header__heading">
        <a class="col-header__link" href="/">clubmate.fi</a>
    </h1>

    <!-- 块元素 -->
    <nav class="nav">

        <!-- 块元素 -->
        <a class="nav__item" href="/">Home</a>

        <!-- 元素与修改器 -->
        <a class="nav__item nav__item--uplink" href="#header">&uarr;</a>

    </nav>
</header>

BEM 是 OOCSS 的加强版，而且利于理解分辨，以后就采用这个啦~
