OC(3) 面对对象编程 (OOP)

2016/9/23 23:16 下午 posted in  ObjC comments  

术语

名称 解释
发送消息 通知对象执行操作称为发送消息. 在[shape draw]中, shape 是对象,剩余部分是操作
类 (class) 一种表示对象类型的结构体,类名采用首字母大写
对象 (Object) 也称为 实例 (instance) 是一种包含值和指向该类的隐藏指针的结构体
实例化 ( instantiation ) 通过类生成一个对象的过程
实例变量 (instance variety) 对象的局部变量
消息 (message) 对象可以执行的操作,通知对象去做什么.对象接收消息后,查询相应的类,找到正确的代码去执行
方法 (method) 相应消息而运行的代码
方法调度 (method dispatcher) OC 的一种机制,用于推测执行什么方法响应特定的消息
接口 (interface) 类为对象提供的特性描述
实现 (implement) 使接口能正常运行的代码

OC 中的 OOP

@interface 部分

创建特定类的对象之前,可以用 @interface 来定义类的公共接口,即 API( Application Programming interface)
以例子说明:

typedef enum
{
 kRedColor,
 kGreenColor,
 kBlueColor
}ShapeColor;

NSString* colorName( ShapeColor color)
{
    switch (color) {
        case kRedColor:
            return (@"kRedColor");
            break;
        case kGreenColor:
            return (@"kGreenColor");
            break;
        case kBlueColor:
            return (@"kBlueColor");
            break;
        default:
            break;
    }
}

typedef struct
{
    int x, y, width, height;
}ShapeRect;

@interface Circle : NSObject
{
    ShapeColor fillColor;
    ShapeRect bounds;
}

- (void) setFillColor: (ShapeColor) fillColor;

- (void) setBounds: (ShapeRect) bounds;

- (void) draw;

@end    //circle
  • @符号是为了防止与 C , C++ 等语言发生冲突
  • { }中表示的是数据成员,每个实例对象都会拥有自己的 fillColor 和 bounds 属性
  • 后面三个方法是函数声明 (method declaration)

中缀符(infix notation) -OC的语法技术

[textThing setStringValue: @"hello" color: kBlueColor];

setStringValue和 color 是方法名称的一部分, @"Hello" 和 kBlueColor 是参数

@implementation 部分

@interface 部分定义了类的公共接口,使对象运行的代码则位于@implementation 当中

  • @implementation 中可以定义在@interface中没有声明过的方法
  • 参数名可以改变
  • 在 OC 中调用方法, 一个叫self的秘密隐藏参数被传递给接收对象, self->fillColor = c 
@implementation Circle

- (void) setFillColor: (ShapeColor) c
{
    fillColor = c ;
}

- (void) setBounds: (ShapeRect) b
{
    bounds = b ;
}

- (void) draw
{
    NSLog(@"drawing a circle at (%d,%d,%d,%d in %@)",
          bounds.x,bounds.y,
          bounds.width,bounds.height,colorName(fillColor));
}
@end

实例化对象

int main(int argc, const char * argv[]) {
    id shapes[3];
    ShapeRect sr1= {2,3,4,4};
    shapes[0]=[Circle new];
    [shapes[0] setFillColor:kGreenColor];
    [shapes[0] setBounds:sr1];
    [shapes[0] draw];   //drawing a circle at (2,3,4,4 in kGreenColor)
    return 0;
}
