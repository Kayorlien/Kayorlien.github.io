初始化顺序、时间、位置

2017/2/18 19:57 下午 posted in  Thinking In Java comments  

顺序

  1. 变量定义的顺序，决定初始化顺序

时间

  1. 静态对象（静态数据）在第一次被创建(访问)时初始化
  2. 静态对象（静态数据）优先于"非静态"对象初始化

位置

  1. 在定义对象的地方
  2. 在类的构造器中
  3. 在即将使用对象之前，这种方式叫做惰性初始化, 这种方法可以减少额外负担，方式类似 如if(member == null) { member = "val"; }
  4. 使用实例初始化
/**
 * 验证初始化顺序
 * 1(static1) -> 3(static3) -> 2(public2) -> 4(public4) -> 5(leg1)
 */
package fun;

public class InitSequence {
    
    public static void init(){
        new Tabel();
    }
}


class Tabel {
    static int static1 = 1; 
    
    public int public2 = 2;
    
    static int static3 = 3;
    
    public int public4 = 4;
    
    static Leg leg1 ;
    
    public Tabel(){
        leg1 = new Leg();
    }
}

class Leg {}
