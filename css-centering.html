css 元素垂直、水平居中

2017/3/22 20:42 下午 posted in  web comments  

水平居中

inline-*

inline-* {
    text-align: center;
}

已知高度的 block 

block {
    margin: 0 auto; 
}

未知高度的 block

block {
    display: inline-block;  
    text-align: center; 
}

垂直居中

inline 单行

inline-* {
    padding-top: 30px;   
    padding-bottom:30px; 
}

inline-* {
    line-height: height;
    /* height 为元素高度 */
}

inline 多行

第一种方法,利用 table 标签

table {
    ...
}
table tr td {
    /* the default value of vertical-align is middle */
}

第二种方法，模拟 table标签，为居中元素添加 vertical-align标签

.parent { 
    display: table;
}

.child {
    display: table-cell;
    vertical-align: middle;
}

已知高度的 block

.parent {
    position: relative;
}

.child {
    position: absolute;
    top:50%;
    margin-top: - height/2;
    /* 偏移是以块的左上角为锚的，以此需减去 block 高度的一半 */
}

未知高度的 block

.parent {
    position: relative;
}

.child {
    position: absolute;
    top:50%;
    transform: translateY(50%);
}

大招，flex

.parent {
    display: flex;
    flex-direction: column;
    justify-content: center;
}
